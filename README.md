# Industrial Surface Defect Detection Dashboard

## 🎯 Overview
This project leverages my 20-year background in visual signal processing to build an automated quality inspection tool. 

## 🚀 Features
- Real-time image upload and analysis
- Contrast-based anomaly detection using OpenCV
- Interactive dashboard built with Streamlit

## 🛠️ Installation & Setup
1. `conda create --name surface-inspection python=3.10`
2. `conda activate surface-inspection`
3. `pip install -r requirements.txt`
4. `streamlit run app.py`
